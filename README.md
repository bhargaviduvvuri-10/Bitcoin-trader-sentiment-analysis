# Bitcoin Trader Behavior & Market Sentiment Analysis

## Objective
Analyze how Bitcoin Fear & Greed sentiment influences trader behavior and performance
using Hyperliquid historical trade data.

## Datasets
- Bitcoin Fear & Greed Index (daily)
- Hyperliquid historical trader data

## Key Insights
- Traders overtrade during Extreme Greed
- Panic selling during Fear leads to losses
- Whales outperform Retail traders in Fear regimes
- Contrarian traders consistently outperform sentiment-driven traders

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Google Colab
